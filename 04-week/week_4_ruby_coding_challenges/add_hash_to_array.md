#ADD HASH TO ARRAY
---
Write a method that accepts two arguments: an array and a hash. Append the contents of the hash to the array.

```
arr = ['a','b','c','d']
hsh = {'e'=>'f','g'=>'h'}

new_array(arr,hsh) #=> returns ["a", "b", "c", "d", "e", "f", "g", "h"]
```

***HINT***

You can do this WITHOUT any looping!
